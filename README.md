## 🎯 Overview

- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- ⚡ Real-time project showcase with animated icons
- 💾 Flask backend with RESTful API
- 📸 Personal gallery section for project documentation
- 🎭 Contextual animations for each IoT project type

**Built with:** Python, Flask, HTML5, CSS3, JavaScript, Three.js

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- pip / conda
- Modern web browser
- macOS / Linux / Windows

### Installation

1. **Clone and navigate**:
```bash
cd /Users/user/Downloads/portofolio
```

2. **Create virtual environment**:
```bash
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. **Install dependencies**:
```bash
pip install -r requirements.txt
```

4. **Run the server**:
```bash
python app.py
```

5. **Open browser**:
```
http://localhost:5001
```

### Using Setup Script
```bash
chmod +x setup.sh
./setup.sh
python app.py
```

## 🌐 API Documentation

### Base URL
```
http://localhost:5001/api
```

### Endpoints

#### 1. GET `/api/projects`
Returns all featured projects with descriptions.
```bash
curl http://localhost:5001/api/projects
```

#### 2. GET `/api/skills`
Returns skills grouped by category with proficiency levels (0-100).
```bash
curl http://localhost:5001/api/skills
```

#### 3. GET `/api/experience`
Returns work history and achievements.
```bash
curl http://localhost:5001/api/experience
```

#### 4. POST `/api/send-message`
Submit a contact form message.
```bash
curl -X POST http://localhost:5001/api/send-message \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com",
    "message": "Great portfolio!"
  }'
```

#### 5. GET `/api/messages`
Retrieve all submitted messages.
```bash
curl http://localhost:5001/api/messages
```

#### 6. GET `/api/stats`
Get portfolio statistics.
```bash
curl http://localhost:5001/api/stats
```

#### 7. GET `/api/health`
Health check endpoint.
```bash
curl http://localhost:5001/api/health
```

## 📸 Gallery Setup

### Adding Personal Photos

1. **Prepare Photos**:
   - Format: JPG/PNG/WebP
   - Size: 600x600px (square recommended)
   - Quality: High resolution, well-lit

2. **Naming Convention**:
   ```
   photo1.jpg - Lab research / Hardware work
   photo2.jpg - Coding session / Development
   photo3.jpg - Team collaboration
   photo4.jpg - Meeting / Presentation
   photo5.jpg - (optional)
   ```

3. **Upload Location**:
   ```
   static/images/personal/
   ```

4. **Auto-Load**: Photos load automatically on refresh

**See [PHOTO_GUIDE.md](PHOTO_GUIDE.md) for detailed instructions.**

## 🎨 Customization

### Personal Information
Edit in `templates/index.html`:
```html
<h1>Your Name</h1>
<p>Your Title</p>
<p>Your Company</p>
```

### Colors
Edit CSS variables in `static/styles.css`:
```css
:root {
    --primary-color: #0a0e27;      /* Main background */
    --accent-blue: #0099ff;        /* Primary accent */
    --text-color: #e0e0e0;         /* Main text */
}
```

### Projects
Edit `app.py` → `get_projects()` function

### Skills
Edit `app.py` → `get_skills()` function
