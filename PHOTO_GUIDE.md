# 📸 Panduan Penempatan Foto Personal

## Lokasi Folder

Silakan letakkan foto-foto Anda di folder:
```
static/images/personal/
```

## Nama File yang Diharapkan

Gunakan nama file sesuai urutan berikut:
- `photo1.jpg` - Foto Lab / Research
- `photo2.jpg` - Coding Session
- `photo3.jpg` - Team Work
- `photo4.jpg` - Meeting & Presentation
- `photo5.jpg` - (optional)
- `photo6.jpg` - (optional)
- `photo7.jpg` - (optional)
- `photo8.jpg` - (optional)

## Format File yang Didukung

- ✅ JPG / JPEG
- ✅ PNG
- ✅ WebP

## Ukuran Foto yang Direkomendasikan

- **Lebar:** 600px - 1200px
- **Tinggi:** 600px - 1200px (Square aspect ratio direkomendasikan)
- **Ukuran file:** < 2MB untuk loading optimal
- **Format:** Landscape atau Square

## Tips untuk Foto yang Bagus

1. **Foto Lab**: 
   - Ambil foto saat sedang bekerja di lab
   - Fokus pada peralatan/hardware IoT yang sedang dikerjakan
   - Pastikan lighting bagus

2. **Coding Session**:
   - Foto saat coding / development
   - Bisa di depan laptop/monitor
   - Tampilkan kedisiplinan dan focus

3. **Team Work**:
   - Foto bersama tim
   - Suasana kolaboratif
   - Bersama colleagues atau mahasiswa

4. **Meeting & Presentation**:
   - Foto saat presentasi project
   - Saat meeting dengan client
   - Tampilkan profesionalitas

## Saran Editing Foto

- Gunakan aspect ratio 1:1 (square) untuk konsistensi
- Pastikan brightness cukup
- Gunakan watermark atau logo personal (optional)
- Pilih foto yang genuine dan authentic

## Automatic Loading

Website akan **automatically load** gambar dari folder `static/images/personal/` dengan nama file sesuai urutan di atas.

Jika foto tidak ada, placeholder dengan icon akan tetap ditampilkan.

## Upload Instructions

1. Letakkan foto ke folder: `static/images/personal/`
2. Refresh website
3. Foto akan otomatis muncul di section "My Journey"

---

**Catatan:** Semua foto akan ditampilkan dengan smooth hover animation dan responsive design.
